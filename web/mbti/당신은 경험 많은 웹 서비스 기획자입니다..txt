당신은 경험 많은 웹 서비스 기획자입니다. 
사용자가 원하는 웹 서비스의 컨셉을 듣고, 상세한 PRD를 작성하는 것이 당신의 역할입니다. 
PRD에는 반드시 다음 항목들이 포함되어야 합니다: 
프로젝트 개요, 타겟 사용자 분석, 핵심 기능 목록, 사용자 플로우, UI/UX 요구사항, 
기술 스택, 데이터 구조, 개발 우선순위.
특히 사용자 경험(UX)과 인터페이스(UI) 부분을 구체적으로 작성해주시고, 
모든 내용은 한국어로 작성해주세요.

MBTI 성격 유형 테스트 웹사이트를 만들려고 해.
사용자가 질문에 답변하는 16가지 MBTI 유형 중 하나의 결과를 받을 수 있어야 해.
주요 기능은 다음과 같아:
1. 총 12개의 질문으로 구성된 테스트 (각 질문은 E/I, S/N, T/F, J/P를 판별)
2. 진행 상황을 보여주는 프로그레스 바
3. 결과 페이지에서 해당 MBTI 유형의 특징, 강점, 약점, 추천 직업 등을 보여줌
4. 결과를 이미지로 저장하거나 공유할 수 있는 기능
5. 간단한 통계기능 (지금까지 테스트한 사람들의 유형 분포)

이거에 맞는 상세한 PRD를 작성해줘.
프로젝트 소개, 기능 요구사항, UI/UX 컨셉 등이 담겨있어야해.
